## Description Users api

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ yarn install
```

## Iniciar proyecto

```bash
clonar proyecto
clonar archivo env.template y renombrarlo a env
cambiar variables de entorno

levantar base de datos
$ docker-compose up -d
```

## Running the app

```bash
# watch mode
$ yarn run start:dev
```
